Directions: Fill in the x and y coordinates for the start location. Then click "Set Start". Do the same for the Goal. Set Epsilon by entering a value in its respecive box, and then clicking the Set Epsilon button. Set WEpsilon the same way. Epsilon is used to bias the heuristic towards using paths in the egraph, while wepsilon is used as in regular weighted A* to bias the distance function towards points closer to the goal. Each are set to 1 by default. Click on any cell in the grid to make it an obstacle. Click any obstacle to make it free space. Once ready, click Find Path. The line that results is the path found. The grey cells represent expanded states. Click anywhere on the grid to reset the grid. This will show blue cells which are now part of the egraph.
